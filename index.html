Обычный стандартный/по умолчанию текст/абзац
b без тега: Привет

Делает текст жирным
mark Привет

i Привет

hr

iframe

Тег используемый для вставки Видео
br Обычный строчный текст: 1,2,3,4,5,6.
Текст с "br" :
1,
2,
3,
4,
5,
6.

Пример с изображением:

C тегом br:

Это блочный тег то есть переносит элементы в другую строку, занимает всю ширину
img

С помощью этого тега ставятся любые изображения/фото/картинки

Image [с английского "изображение"]

Его вспомогательный тег "li"

Элемент представляет собой упорядоченный список элементов — обычно отображается как нумерованный список.
ul
  • Blue cheese
  • Black cheese
  • yellow cheese

Его вспомогательный тег "li"

Элемент представляет собой неупорядоченный список элементов, обычно отображаемый в виде маркированного списка.
